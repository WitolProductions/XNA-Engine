Xna-engine

An open source game engine built using C# with the XNA framework. Right now it includes a fully functional Input Handler that not only works on Windows, but, also works on Windows Phone and with the Xbox and Xbox 360. Support is included for Mouse, Keyboard, Controllers, String Generator (Keyboard Typing), and Windows Phone Input and Sensors. Work has begun on the Graphical systems that the engine will make use of.
